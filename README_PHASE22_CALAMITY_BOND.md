# WILDS Phase 22：災禍平衡與老皮連攜

## 玩家能直接感受到的更新

- 世界災禍改為祭壇周圍 620px 的正式戰場，地面會顯示邊界。
- 玩家離開戰場、死亡或回到紀錄點時，光彈、隕石、深淵、眼球、骨刺與震波會立即清除。
- 災禍不再跨地圖鎖定玩家；重新踏入戰場有 2.4 秒安全前搖。
- 正式重生至少有 2.8 秒共同無敵，避免剛睜眼就被舊攻擊再次擊倒。
- 首輪巨鯤生命由 1800 降為 1500，泰坦由 2500 降為 2050；彈幕密度、接觸傷害、隕石、骨刺、震波與飛升倍率同步調整。
- 每次轉階會清除舊階段攻勢並留出 1.35 秒辨識時間。

## 戰鬥、秘境與老皮的連結

- 玩家命中：連攜 +8。
- 玩家與老皮在 1.25 秒內命中同一區域：額外連攜 +12。
- 完美閃避：連攜 +14。
- 老皮成功擋傷：連攜 +18。
- 連攜滿 100 自動發動「彈力夾擊」，造成 26 傷害並回復 18 老皮守護能量。
- 廢墟／洞穴現在顯示為廢墟秘境／洞穴秘境；開啟前，老皮會在近距離主動發現並提醒。
- 搜刮加成依當下連攜分成三級；秘境開啟後獲得 45 秒共鳴，期間連攜累積為 1.6 倍，夾擊傷害提高為 34。

這形成雙向循環：先戰鬥可提高秘境收益，探索秘境又能強化下一場共鬥。

## 平衡與防濫用

- 主動撤離會讓災禍回復 8% 最大生命，死亡撤離回復 10%。
- 退場清彈幕是安全機制，不是免費重置首領進度。
- 連攜與秘境共鳴是暫態戰鬥狀態，不寫入存檔，避免舊存檔遷移與讀檔刷增益。
- 存檔格式維持 v20，v51 與 v50 的所有進度可直接沿用。
- 若 v51 存檔正停在災禍戰中，讀檔時會保留原生命百分比並套用新版最大生命，不要求重新開戰。

## 自動驗證

`node tools/check-phase22.js` 會實測戰場邊界、攻勢清除、撤離代價、重新進場前搖、死亡清場、重生安全期、彈力夾擊與秘境共鳴。

完整檢查仍使用：

```bash
node tools/check-all.js
```
