# WILDS Phase 24：老皮與荒野生活

## 玩家能直接感受的更新

- 哇塞秘境不再固定走同一套流程；每座入口會從伸縮橋、彈簧、開鎖、黑暗嗅聞、雙人守點中決定性抽出三間房。
- 深層挑戰會出現「狼群回音、沉重腳步、飢餓詛咒」等預兆，玩家能在深入前看見風險。
- 終極三明治把營火料理、夜間採集、勇闖秘境與區域首領串成一條長期任務。
- 老皮會依玩家實際行為形成「冒險狂、安心守護、三明治、彈力」四種搭檔章節，不使用額外經驗條。
- 野外長途移動、黑夜休息與受到威脅時，老皮會自動變成座騎、臨時庇護所或把玩家拉離攻擊方向。

## 多變哇塞秘境

- 每次只進行三間房，入口種子相同時配置固定，讀檔無法洗房間。
- 黑暗嗅聞會縮小視野並留下老皮氣味腳印。
- 雙人守點會由老皮撐住石門，玩家負責清除守衛。
- 第一波完成後仍可安全離開，或依預兆挑戰深層守衛。
- 「稀有秘境」三明治效果會把下一座秘境的第三間房替換為共鳴藏寶房。

## 終極三明治

四種任務材料：

1. 特製麵包：完成三次營火料理。
2. 月光香草：夜間採集兩次漿果或夜光蘑菇。
3. 秘境醬汁：完成一次深入秘境。
4. 傳說配料：與老皮共同擊敗區域首領或世界災禍。

材料解鎖後，每天可在營火旁使用烤肉、肉乾與漿果製作一次，並由玩家選擇：

- 立刻補滿老皮守護能量。
- 下一次老皮戰衣由 6 秒延長為 8 秒。
- 今天第一次倒地由終極三明治救援。
- 下一座秘境必定出現稀有藏寶房。

## 行為羈絆與營火生活

- 勇闖秘境與擊敗首領會形成「冒險狂搭檔」。
- 擋傷、救援與安全撤退會形成「安心守護搭檔」，守護消耗由 40 降為 34。
- 在營火、熔爐或睡袋旁分享食物會形成「三明治搭檔」。
- 完美閃避會形成「彈力拍檔」，額外增加 6 點連攜。
- 共同睡到天亮時，老皮會依最近的冒險方式回顧當天，而不是播放固定台詞。

## 存檔與快取

- 存檔格式升為 v22；v21 與更舊存檔會逐版遷移，不會清空既有進度。
- 秘境中途房間、即時座騎與庇護所屬於暫態；只有共同經歷、三明治任務與羈絆章節寫入存檔。
- Service Worker 快取版本為 `wilds-v54-laopi-wilderness-life`。

## 自動驗證

`node tools/check-phase24.js` 會實測三明治素材、四段任務、每日限制、四種效果、行為羈絆、多變房間、稀有房與 v21→v22 遷移。

完整檢查：

```bash
node tools/check-all.js
```
