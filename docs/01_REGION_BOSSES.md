# 五大區域首領

## 1. 森林王：終焉九頭蛇
- 區域：森林／毒沼交界
- 建議 HP：900
- 階段：100–70%、70–35%、35–0%
- 技能：
  - 毒液散射：朝玩家扇形發射 5 發，落地生成 6 秒毒池。
  - 蛇首橫掃：近距離 120°掃擊，擊退玩家。
  - 蛇群召喚：召喚 3–6 隻小蛇。
  - 再生蛇首：每進入新階段增加一條攻擊軌道與 12% 攻速。
  - 狂暴毒雨：35% 以下週期性落下毒彈。
- 掉落：神盾。

## 2. 雪原王：冰霜聖龍
- 區域：雪原
- 建議 HP：1050
- 技能：冰霜吐息、追蹤冰柱、飛天俯衝、暴風雪領域。
- 特殊：暴風雪期間玩家移速降低，但火堆附近可免疫減速。
- 掉落：神槍。

## 3. 沼澤王：腐敗巨像
- 區域：沼澤
- 建議 HP：1250
- 技能：巨大泥拳、腐敗噴流、毒球拋射、小型黏液分裂。
- 特殊：場地會逐漸被毒泥覆蓋，攻擊巨像背部結晶可清除部分毒泥。
- 掉落：神斧。

## 4. 天空王：雷霆天鷹
- 區域：高地／天空遺跡
- 建議 HP：850
- 技能：高速俯衝、雷羽散射、落雷標記、旋風牆。
- 特殊：俯衝後短暫落地，是主要輸出窗口。
- 掉落：神翼。

## 5. 火山王：熔岩魔神
- 區域：火山
- 建議 HP：1450
- 技能：岩漿拳、火柱陣、熔岩隕石、地面裂解。
- 特殊：站在冷卻岩地安全；場地會循環改變安全區。
- 掉落：神劍。

## 共通規則
- 首次擊敗才掉落神武；重複擊敗掉落強化材料。
- Boss 離開領域後返回出生點並逐步恢復生命。
- Boss 擁有 melee、projectile、zone、summon 四類標準技能介面。
