# 世界級災禍

## 解鎖條件
- 世界天數 >= 20。
- 至少持有 1 件神武。
- 玩家前往古代祭壇並確認召喚。

## 1. 深淵巨鯤
- 第一階段：暗潮撞擊、眼球射線、深淵小怪。
- 第二階段：黑洞吸引、吞噬前搖、漂浮碎石。
- 第三階段：升空、隕石雨、全場紫色掃射。
- 獎勵：深淵之神 Skin、深淵神格。

## 2. 骷髏泰坦
- 第一階段：巨拳、地震波、骨刺線。
- 第二階段：亡靈軍團、骨牢、投擲巨石。
- 第三階段：拆下骨臂作為巨槌，攻擊範圍提升。
- 獎勵：死亡君王 Skin、亡靈神格。

## 3. 星辰古神
- 技能：重力井、時間減速區、星辰雷射、位置交換。
- 獎勵：星神 Skin、星辰神格。

## 4. 終焉鳳凰
- 技能：火羽雨、環形火牆、俯衝爆炸。
- 特殊：首次歸零後重生，第二條生命技能加速。
- 獎勵：不死鳥 Skin、鳳凰神格。

## 5. 終焉之眼
- 最終戰，混合已擊敗首領技能，但每次只啟用兩種技能組，避免不可讀。
- 階段切換會破壞場地並改變安全區。
- 獎勵：終焉 Skin、終焉神格、New Game+。

## 災禍戰共通規範
- 戰鬥前 10 秒倒數，可取消召喚。
- 戰鬥時禁止睡覺與快速旅行，但允許使用已建造設施。
- 玩家死亡後災禍回滿血，召喚材料不消耗或僅損失少量，避免挫折過大。
