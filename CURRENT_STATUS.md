# WILDS 目前狀態

目前正式開發版本：**Phase 19**  
存檔格式：**v19**  
Service Worker：**wilds-v43**

## 已完成

- 五個區域首領：九頭蛇、冰霜聖龍、腐敗巨像、雷霆天鷹、熔岩魔神。
- 兩個守衛 Boss、四名可招募夥伴、五件可視化神武與共鳴。
- 萬眼巨鯤、骸骨泰坦、飛升輪迴與神格碎片。
- 兩名角色各五套專用 Skin，共十張動畫表。
- 區域／祭壇導航、快速移動、可視化主手與神武裝備。
- 冒險日誌、荒野榮譽、Skin 收藏與實體階級外觀獎勵。
- 祭壇十秒召喚與取消、攻擊預警、命中停頓、鏡頭震動、傳送特效。
- 體力翻滾、0.28 秒無敵、手機翻滾按鈕與鍵盤空白鍵操作。
- 區域首領／世界災禍的真實技能前搖、轉階提示與實體紫金光環。
- 裝置端遊戲設定：震動、閃光、傷害數字、鏡頭震動、音量與省電模式。
- 雲端覆蓋前的本機／雲端進度摘要比較與明確選擇。
- 傳說 Skin 延遲到查看或裝備時才下載並快取，降低 PWA 首次安裝負擔。
- 完美閃避：早期翻滾成功返還 12 體力，下一擊增加 18 傷害並有專用回饋。
- 減少動態效果、高對比警示、受傷閃光控制與音量／靜音狀態同步。
- 全螢幕面板會暫停世界模擬並鎖定操作，避免整理裝備時遭受攻擊。
- PWA 更新先待命，玩家完成存檔並確認後才安全切換版本。
- 一般素材分階段載入與 runtime cache，縮短首次安裝與開局等待。
- 四名夥伴各自具備待機、移動、攻擊三幀透明動畫表，已移除舊素材的白色貼紙邊與去背殘留。
- 夥伴改用隨玩家面向旋轉的隊形、加速度跟隨、平滑轉向、呼吸／步伐形變與獨立攻擊距離。
- 騎士斬擊、弓手射線、小貓爪痕與精靈魔法具有各自的攻擊幀和畫面回饋。
- 本機 v19 存檔可匯出 JSON 備份，匯入時會先校驗再覆蓋。
- IndexedDB 主存檔＋兩份輪替備份＋校驗＋自動復原＋雲端同步。
