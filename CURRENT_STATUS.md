# WILDS 目前狀態

目前正式開發版本：**Phase 25（野獸生態與顯示修復）**  
存檔格式：**v22**  
Service Worker：**wilds-v55-wild-ecology**

## 已完成

- 五個區域首領：九頭蛇、冰霜聖龍、腐敗巨像、雷霆天鷹、熔岩魔神。
- 兩個守衛 Boss、四名可招募夥伴、五件可視化神武與共鳴。
- 萬眼巨鯤、骸骨泰坦、飛升輪迴與神格碎片。
- 兩名角色各六套專用 Skin，共十二張動畫表。
- 區域／祭壇導航、快速移動、可視化主手與神武裝備。
- 冒險日誌、荒野榮譽、Skin 收藏與實體階級外觀獎勵。
- 祭壇十秒召喚與取消、攻擊預警、命中停頓、鏡頭震動、傳送特效。
- 體力翻滾、0.28 秒無敵、手機翻滾按鈕與鍵盤空白鍵操作。
- 區域首領／世界災禍的真實技能前搖、轉階提示與實體紫金光環。
- 裝置端遊戲設定：震動、閃光、傷害數字、鏡頭震動、音量與省電模式。
- 雲端覆蓋前的本機／雲端進度摘要比較與明確選擇。
- 傳說 Skin 延遲到查看或裝備時才下載並快取，降低 PWA 首次安裝負擔。
- 完美閃避：早期翻滾成功返還 12 體力，下一擊增加 18 傷害並有專用回饋。
- 減少動態效果、高對比警示、受傷閃光控制與音量／靜音狀態同步。
- 全螢幕面板會暫停世界模擬並鎖定操作，避免整理裝備時遭受攻擊。
- PWA 更新先待命，玩家完成存檔並確認後才安全切換版本。
- 一般素材分階段載入與 runtime cache，縮短首次安裝與開局等待。
- 四名夥伴各自具備待機、移動、攻擊三幀透明動畫表，已移除舊素材的白色貼紙邊與去背殘留。
- 夥伴改用隨玩家面向旋轉的隊形、加速度跟隨、平滑轉向、呼吸／步伐形變與獨立攻擊距離。
- 騎士斬擊、弓手射線、小貓爪痕與精靈魔法具有各自的攻擊幀和畫面回饋。
- 老皮為開場常駐的第五位特殊夥伴，不佔四名招募夥伴名額，也不消耗日常食物。
- 老皮具備待機、伸縮移動、護主擋傷三幀透明動畫，素材失敗時會退回向量版本。
- 守護鎖定優先處理正在威脅玩家的敵人；普通怪、Boss、災禍共用同一傷害結算順序。
- 神武先於老皮攔截傷害；毒池、熔岩區與深淵持續傷害不會浪費老皮守護冷卻。
- 老皮以 40 守護能量吸收一次攻擊，16 秒冷卻；能量會依戰鬥狀態恢復並顯示於 HUD。
- 不死鳥未觸發時，老皮每天可用滿能量和熟食／肉乾／莓果進行一次瀕死救援。
- 共同過夜、擋傷、首領／災禍勝利與救援記憶會保存，安全時可觸發回呼台詞。
- 完成同行 3 天、擋傷 8 次並共同擊敗首領後，解鎖純外觀 Skin「找到的家人」。
- 右上角常駐操作由十顆圖示縮減為「導航＋功能」兩顆具名按鈕，其餘功能集中到快速面板。
- 快速面板提供移動、裝備、換角色、日誌、獎勵、設定、音效、雲端與診斷，選擇後自動收起。
- 收合後的功能鍵會同步顯示待領獎勵紅點；音效與雲端圖示會反映目前狀態。
- 座標、區塊與 FPS 不再常駐遮擋畫面，仍可由診斷面板查看完整資訊。
- HUD 與快速面板支援瀏海安全區、窄螢幕與低高度橫向配置；開啟選單時會暫停世界並鎖定移動。
- 災禍戰限制在祭壇周圍 620px 戰場；離場、死亡與回紀錄點時會清除殘留彈幕，首領不再跨地圖追擊。
- 撤離會讓災禍回復 8% 生命，重新進場有 2.4 秒安全前搖；死亡則回復 10%，避免用退場無成本重置攻勢。
- 首輪巨鯤／泰坦血量、攻擊數量、傷害與頻率已下修；飛升生命與傷害倍率也改為較平滑成長。
- 災禍轉階會清除上一階段殘留招式並提供 1.35 秒轉階緩衝，畫面顯示戰場邊界與安全前搖。
- 正式重生提供至少 2.8 秒共同無敵，老皮／不死鳥瀕死救援會清除災禍殘留招式並取得短暫喘息。
- 玩家命中、老皮跟擊、完美閃避與老皮擋傷會累積連攜；滿值改為玩家主動發動，不再自動浪費。
- 連攜就緒時才出現臨時按鈕：短按發動彈力夾擊，長按發動六秒「老皮戰衣」。
- 老皮戰衣強化伸縮拳距離與傷害，翻滾變成彈力衝撞，可擋一次重擊，並持續消耗守護能量。
- 廢墟與洞穴升級為真正的雙人哇塞秘境：嗅聞探路、變橋、變彈簧、解鎖、雙人戰鬥與寶藏抉擇。
- 秘境第一波完成後可安全離開，或挑戰更強共鳴守衛取得更多獲勵；完成與深入次數會記錄。
- 新增老皮四格透明變形素材：嗅聞、變橋、彈簧與戰衣；素材失敗時仍有向量回退。
- HUD 顯示守護、連攜、共鳴倒數及秘境當前目標；連攜按鈕平常不佔畫面。
- 哇塞秘境改為每座入口決定性抽取三間房，新增黑暗嗅聞、雙人守點、稀有藏寶與深層預兆。
- 終極三明治以營火料理、夜間採集、深入秘境與首領戰解鎖四種材料；每天可製作一次並主動選擇效果。
- 老皮依玩家的勇闖、守護／撤退、分享食物與完美閃避形成四種搭檔章節，不增加常駐經驗條。
- 安心守護章節降低擋傷能量消耗；彈力拍檔提高完美閃避連攜；其他章節會改變秘境與營火回饋。
- 長途移動時老皮可短暫變成彈力座騎；夜間無庇護時會撐成臨時帳篷；擋傷時可把玩家拉離攻擊方向。
- 在營火、熔爐或睡袋旁可和老皮分享食物；共同過夜會依最近冒險方式產生不同回顧。
- 新增 160×160 透明終極三明治遊戲圖示，並納入 PWA 快取。
- 修正野豬、熊、烏鴉因缺少素材顯示高度而可能完全畫不出來；原有六種野獸現在都具備有效高度、待機／移動素材與向量備援。
- 新增狐狸、山羊、巨獾及六張正式透明素材；狐狸夜間迂迴撲擊、山羊預警後直線衝撞、巨獾受擊後長時間追擊。
- 十八張實體野獸素材已納入預載與離線快取；寬幅動作使用加大畫面剔除邊界，避免螢幕邊緣突然跳出／消失。
- 素材底層增加無效高度保護；未來新增生物即使漏填高度，也會退回安全尺寸，不會再變成隱形野獸。
- 老皮與四名夥伴不再自動攻擊和平野獸；快速移動與夥伴鎖定共用威脅判斷，擊殺訊息也會顯示實際掉落。
- 本機 v21 與更舊存檔可自動升級 v22；存檔可匯出 JSON 備份，匯入時會先校驗再覆蓋。
- IndexedDB 主存檔＋兩份輪替備份＋校驗＋自動復原＋雲端同步。
