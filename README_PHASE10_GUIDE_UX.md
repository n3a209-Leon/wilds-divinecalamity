# WILDS Phase 10：探索導航與祭壇召喚提示

## 本次新增

- 玩家頭頂常駐方向箭頭，預設指向最近的存活區域首領。
- 箭頭附目標名稱與即時距離；抵達範圍後改為綠色勾號。
- 右上角 `🧭` 可依序切換五位區域首領與世界災禍祭壇。
- 被追蹤的首領死亡後，自動改指向下一位存活區域首領；全部擊敗後改指向祭壇。
- 祭壇提示直接顯示解鎖條件：第 20 天且至少取得一件神武。
- 祭壇開啟後顯示完整操作：走近祭壇，按右下角 `☄️` 召喚。
- 災禍召喚後，導航自動轉向目前的世界災禍。

## 相容性

- 不變更存檔資料格式，仍為 v14，舊存檔可直接讀取。
- `sw.js` 快取升為 `wilds-v34`，新增 `js/guide.js` 離線快取。
- Phase 9 的熔岩魔神、五位區域首領、神武與素材清單均未改動。
