# WILDS Phase 21：精簡操作介面

## 玩家能直接感受到的更新

- 右上角原本固定顯示十顆圓形按鈕，現在只保留「導航」與「功能」兩顆具名按鈕。
- 點「功能」會展開九宮格快速面板：移動、裝備、換角色、日誌、獎勵、設定、音效、雲端、診斷。
- 點選功能後面板會自動收起；也能點背景、關閉鍵或鍵盤 Esc 收起。
- 待領獎勵紅點會同步顯示在收合後的「功能」鍵，不必展開才能發現獎勵。
- 音效關閉時會顯示「音效關」，雲端登入後會顯示「已同步」。
- 座標、區塊與 FPS 改由診斷面板查看，不再長期遮住遊戲畫面。
- 左上狀態欄縮小並加上半透明底板，生命、飽食、體力、理智、老皮與神武狀態仍保留。

## 手機版配置

- 直向畫面使用 3×3 快速面板，每個項目都有圖示與文字，降低只看 emoji 猜功能的負擔。
- 低高度／橫向畫面自動改為五欄，避免面板過高壓住操作區。
- HUD、選單與操作鍵皆計入 iPhone 瀏海及底部安全區。
- 快速面板展開時暫停世界模擬並鎖定移動，整理功能時不會被敵人偷襲。

## 相容性

- 存檔格式維持 v20，v50 的老皮、Skin、獎勵與所有進度不需要遷移。
- 所有既有按鈕 ID 與功能入口均保留，原有模組與自動測試仍可沿用。
- Service Worker 快取更新為 `wilds-v51-compact-ui`，部署後會走現有的安全更新流程。

## 自動驗證

執行：

```bash
node tools/check-all.js
```

Phase 21 會驗證常駐按鈕數量、九項快速功能、無重複 DOM ID、獎勵紅點同步、選單開關與自動收合、輸入鎖定、響應式五欄配置及 v51 快取版本。
