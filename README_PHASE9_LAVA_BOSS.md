# Phase 9：第五位區域首領「熔岩魔神」

本階段以 Phase 8 完整版為基準，加入正式的第五位區域首領，沒有重寫既有四位首領與五神武系統。

## 圖片素材

- `assets/boss_lava.png`：熔岩魔神待機姿勢，320×294，透明背景。
- `assets/boss_lava_atk.png`：熔岩重拳攻擊姿勢，320×253，透明背景。
- 保留原圖的岩石結構、熔岩裂紋、火焰、碎石與攻擊煙塵；兩張圖皆以腳底為錨點等比繪製。

## 戰鬥設計

- HP：900，三階段，25% 以下進入第三階段。
- 岩漿拳：近距離重擊，攻擊期間使用 `_atk` 圖。
- 熔岩彈：朝玩家扇形發射，階段越高彈數越多。
- 火柱陣：地面先出現虛線預警圈，延遲後爆發；所有攻擊都有可反應前搖。
- 地面裂解：產生持續傷害熔岩區，第三階段會同時形成多區域。
- 冷卻岩地：戰鬥中週期性出現藍灰色安全區，站在其中可免疫熔岩彈、岩漿拳、火柱與熔岩地面傷害。
- 離開領域後會回到出生點，不會追逐玩家穿越整張地圖。
- 首次擊敗獎勵神劍；若神劍已由暗影魔解鎖，會顯示「神武已持有」而不重複解鎖。

## 系統整合

- `art.js` 素材登錄由 91 項增加至 93 項。
- Service Worker 升至 `wilds-v33`，兩張新圖已加入逐筆容錯快取。
- 擊敗狀態沿用既有 `bosses` 動態鍵值存檔，舊 v14 存檔不需要破壞性遷移；舊玩家載入後熔岩魔神會正常生成。
- `render.js` 加入熔岩彈、熔岩區、火柱前搖／爆發與冷卻岩地顯示。

- 首次擊敗可解鎖神劍，重複獎勵不會覆蓋既有神武資料。
