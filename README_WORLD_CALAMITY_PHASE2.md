# 世界災禍 Phase 2

本階段加入第一位可實際戰鬥的世界級災禍「萬眼巨鯤」。

## 新增玩法
- Day 20 且持有至少一件神武後，世界災禍祭壇甦醒。
- 玩家靠近祭壇並按攻擊鍵，正式召喚萬眼巨鯤。
- 三階段攻擊：萬眼彈幕、深淵黑洞吸引、隕石預警與爆炸、深淵眼球召喚。
- 近戰與箭矢都能鎖定、追蹤並傷害萬眼巨鯤與深淵眼球。
- 擊敗後解鎖並自動裝備 Skin「深淵之神」。
- 深淵之神每 12 秒觸發一次深淵脈衝，攻擊玩家附近敵人。
- 存檔升級到 v9，保存災禍血量、擊敗狀態、Skin 收藏與裝備。
- Service Worker 快取升級至 wilds-v26。

## 防護
所有新模組呼叫都保留存在性守衛；舊測試未載入 skins.js 或 calamity.js 時不應因此崩潰。
