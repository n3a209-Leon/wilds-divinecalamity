# WILDS Phase 23：老皮變形大冒險

## 玩家能直接感受的更新

- 廢墟與洞穴不再是按一下就領獲勵，而是由玩家與老皮一起破解的短型秘境。
- 老皮會嗅聞探路、伸長變橋、捲曲變彈簧，並伸入古老機關解鎖。
- 破解機關後會進入雙人戰鬥；第一波結束後可選擇安全離開，或繼續挑戰共鳴守衛。
- 成功完成後才寫入秘境搜刮記錄；中途死亡或離開會清掉守衛並安全重置。

## 主動連攜

- 連攜值滿 100 後不再自動發動，避免打在小怪或戰鬥結束時浪費。
- 只有就緒時才顯示臨時連攜按鈕，不增加平常的 UI 負擔。
- 短按：對附近威脅發動「彈力夾擊」，秘境共鳴期間傷害更高。
- 長按：發動六秒「老皮戰衣」。

## 老皮戰衣

- 伸縮拳增加 34 攻擊距離與 12 傷害。
- 翻滾變成彈力衝撞，接觸敵人時造成傷害。
- 戰衣可吸收一次重擊，第二次攻擊會正常結算。
- 發動期間持續消耗守護能量；時間或能量結束後老皮會進入短暫疲勞。

## 存檔與快取

- 存檔格式升為 v21，v20 會自動補上秘境完成、深入挑戰與戰衣發動記錄。
- 共鳴、戰衣秒數與中途秘境屬於暫態，重開時安全重置，避免讀取到一半的關卡。
- Service Worker 快取版本為 `wilds-v53-laopi-adventure`。

## 自動驗證

`node tools/check-phase23.js` 會實測四段變形機關、戰鬥波次、寶藏抉擇、主動連攜、老皮戰衣、素材透明通道與 v20→v21 遷移。

完整檢查：

```bash
node tools/check-all.js
```
