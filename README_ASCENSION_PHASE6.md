# WILDS — 飛升輪迴 Phase 6

完成兩位世界級災禍後，世界災禍祭壇會轉化為「飛升祭壇」。

## 新增玩法

- 擊敗萬眼巨鯤與骸骨泰坦後，祭壇可持續召喚強化版災禍。
- 飛升災禍依序在巨鯤與泰坦之間輪替。
- 每完成一輪：
  - 飛升等級 +1
  - 神格碎片 +1
  - 下一位災禍生命與傷害提高
- 每枚神格碎片永久提高神武與 Skin 傷害 6%。
- 永久加成最多計算 10 枚，避免數值失控。

## 顯示

- 祭壇顯示目前神格碎片數量。
- 飛升戰鬥顯示飛升等級。
- 擊敗後顯示輪迴完成與永久強化提示。

## 存檔

- 存檔版本升級至 v13。
- 保存飛升輪迴、神格碎片、下一位災禍及進行中的飛升戰鬥。
- v12 舊存檔自動遷移。
