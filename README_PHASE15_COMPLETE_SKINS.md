- 星神：平定五個區域並集齊五件神武。

- 星神每 20 秒展開 3 秒星界，使附近普通敵人與區域首領移速降低 55%。

- 終焉讓所有神武冷卻縮短 10%，代價是夜間承受敵人傷害提高 10%。
