# WILDS Phase 11：快速移動與真實裝備

## 快速移動

- 右上角新增 `🌀` 快速移動按鈕。
- 目前營地永久可返回；移動睡袋改設營地後，目的地會同步更新。
- 五個區域首領領地需親自進入探索範圍，或擊敗該首領後永久解鎖。
- 世界災禍祭壇需親自探索，或達成「第 20 天＋任一神武」後解鎖。
- 尚未擊敗的首領只會傳送到領地外圍；已擊敗後可直接抵達領地中心。
- 首領、世界災禍或敵對生物在附近時禁止傳送，抵達後有 3 秒穩定時間。
- 已探索地點寫入 v15 存檔，雲端與本機都會保存。

## 真實裝備欄

- 右上角新增 `🛡️` 角色裝備按鈕。
- 主手工具只能裝一件：徒手、石斧、金屬斧、石鎬、金屬鎬或木弓。
- 只有目前主手會提供攻擊、採集或射箭效果，未裝備工具不再於後台被動生效。
- 五件神武可個別裝備與卸下；只有已裝備神武會發動能力與共鳴。
- 玩家身上會常駐顯示主手、神盾、神槍、神劍、神斧與神翼。
- 神翼本體常駐背部，發動時加亮；神盾、主手工具會跟隨角色朝向與揮擊。
- 舊 v14 存檔載入時會自動選擇最高級近戰工具作為主手，既有神武裝備狀態完整保留。

- Service Worker：`wilds-v35`。
- Phase 9 熔岩魔神、Phase 10 導航、93 項素材與既有 Boss 攻擊幀均保留。
