# WILDS Phase 20：老皮守護夥伴

## 玩家能直接感受到的更新

- 老皮從開場起永久同行，不需要招募，也不會占用原本四名夥伴名額。
- 老皮會跟隨玩家面向換位、伸縮奔跑，並優先攻擊正在追逐或準備命中玩家的敵人。
- 玩家即將受到可攔截攻擊時，老皮會伸長身體變成護盾，吸收一次完整傷害。
- HUD 會顯示老皮的守護能量；擋傷消耗 40 點，冷卻 16 秒，不能無限免傷。
- 若玩家瀕死且不死鳥沒有發動，滿能量的老皮每天可消耗一份烤肉、肉乾或漿果救援一次。
- 老皮平時大多安靜，危險時會收起玩笑；第一次共同過夜、擊敗首領、平息災禍與救援會形成持久記憶。
- 達成羈絆條件後，可解鎖兩位玩家各自適用的「找到的家人」純外觀 Skin。

## 戰鬥規則

傷害結算順序固定為：

1. 翻滾無敵與完美閃避。
2. 神盾／神翼。
3. 老皮守護。
4. Skin 傷害倍率。
5. 正式扣除生命。

毒池、熔岩區、冰霜／腐敗地板與深淵持續傷害不觸發老皮擋傷；近戰、投射物、隕石、骨刺與震波可觸發。

## 羈絆與存檔

存檔格式由 v19 升為 v20，新增 `bondMate`：

- 是否已相遇與記憶旗標。
- 同行天數。
- 共同擊敗首領／災禍次數。
- 擋傷、救援與回呼日期。

心情、當前能量、動畫與冷卻屬於短暫狀態，不寫入存檔。v19 本機或雲端存檔會自動補上安全預設值；清除存檔與世界種子更換也會正確重置。

「找到的家人」解鎖條件：同行 3 個完整遊戲日、成功擋傷 8 次，並共同擊敗任一區域首領或世界災禍。Skin 為純外觀，不疊加新的戰鬥能力。

